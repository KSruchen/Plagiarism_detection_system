# Plagiarism Detection System

## Overview

A web-based plagiarism detection application developed using Python and Flask. The system allows users to upload PDF documents, extract textual content using PDF parsing and OCR techniques, compare document content, and generate a similarity score.

## Features

* Upload and compare two PDF documents
* Text extraction using PDFPlumber
* OCR support for scanned PDFs using Tesseract OCR
* Similarity score calculation
* Extracted text visualization
* Simple web-based interface
